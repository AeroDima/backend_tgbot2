# Telegram Calorie Bot (SQLite + Gemini AI)

Бот для підрахунку калорій з використанням штучного інтелекту для аналізу їжі.

## Можливості

- 💾 **SQLite**: Всі ваші дані (профіль, записи їжі) зберігаються в базі даних.
- 🤖 **Gemini AI**: Автоматичне визначення калорій з текстового опису страви.
- 📊 **Статистика**: Перегляд денної норми та фактичного споживання.

## Налаштування

1. Скопіюйте `.env.example` в `.env`:
   ```bash
   cp .env.example .env
   ```
2. Вставте ваші `BOT_TOKEN` та `GEMINI_API_KEY` у файл `.env`.

## Встановлення та запуск

1. Встановіть залежності:
   ```bash
   npm install
   ```
2. Запустіть бота:
   ```bash
   npm start
   ```

## Команди

- `/start` - Початок роботи та інструкції.
- `/set_profile` - Налаштування профілю (вік, зріст, вага, стать, активність).
- `/my_profile` - Перегляд збережених даних.
- `/add_meal` - Додати прийом їжі (наприклад: "борщ 300г та шматок хліба").
- `/today` - Список зʼїденого за сьогодні та сума калорій.
