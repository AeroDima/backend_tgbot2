# Telegram Calorie Bot

Бот для підрахунку калорій (BMR та TDEE) за формулою Міффліна-Сан Жеора.

## Налаштування

1. Скопіюйте `.env.example` в `.env`:
   ```bash
   cp .env.example .env
   ```
2. Вставте ваш `BOT_TOKEN` у файл `.env`.

## Встановлення та запуск

1. Встановіть залежності:
   ```bash
   npm install
   ```
2. Запустіть бота:
   ```bash
   npm start
   ```

Для розробки (з автоперезапуском):
```bash
npm run dev
```

## Команди

- `/start` - Початок роботи та інструкції.
- `/set_profile` - Налаштування профілю (вік, зріст, вага, стать, активність).
- `/my_profile` - Перегляд збережених даних та розрахованих показників.
